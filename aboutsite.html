Wow, what a great website.

GUI:

Click on a folder in the files window to expand the directory. The site is roughly designed to imitate a linux file browser. Files are kind of organized by directory.

CLI:

Click the terminal icon in the top right corner to use the terminal. This terminal supports the following commands:

  • ls -Lists files and folders in the current directory
  • cd -Navigates into a folder (e.g, cd about)
  • cat -Opens an HTML document corresponding to that file
  • clear -Clears the terminal
*Note that CLI is only available on desktop version of this site. This is because managing a virtual keyboard alongside GUI and terminal on smaller screens would be inconvenient. The GUI remains the default experience, but viewers on larger screens can also use CLI if they prefer a terminal like interaction.
