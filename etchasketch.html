Etch a sketch

This project simulates the core functionality of the Etch A Sketch toy by providing an interactive canvas where the user can draw lines and shapes. Instead of physical knobs, users have the ability to draw using mouse/trackpad movements or interactions with a touch screen, giving them a virtual version of the original Etch A Sketch. The project emphasizes simple user interface design, responsiveness, and interactive drawing.
